# abolish-us

A TanStack Start and React application.

## Quickstart

```bash
bun install
bun --bun run dev
```

## Scripts

```bash
bun --bun run dev
bun --bun run build
bun --bun run preview
bun --bun run test
bun --bun run lint
bun --bun run format
bun --bun run check
bun --bun run storybook
bun --bun run build-storybook
```

## Project Standards

- Package manager: `bun`
- Language: TypeScript
- Framework: TanStack Start + React
- Linting and formatting: Biome
- Component docs: Storybook
